This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Run the application

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Run the json server

```bash
npx json-server db.json --port 4000
# port 4000 to differenciate from the default of 3000, as thats where the app is running
```

Open [http://localhost:4000](http://localhost:4000) to see the db.json files content in the browser.
